# 📚 Library Management System in C

This is a simple **Library Management System** written in **C**.  
It uses **file handling** to store and manage books in a binary file (`Book.dat`).

---

## ✨ Features
- ➕ Add a new book  
- 📖 Display all books  
- 🔍 Search book by ID  
- 📦 Issue a book  
- 🔄 Return a book  
- ❌ Exit  

---

## 🛠️ Technologies Used
- C Programming Language
- File Handling in C
- Structures
